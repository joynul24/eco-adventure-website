# 🌿 EcoAdventure

EcoAdventure is a modern eco-friendly web platform where users can explore sustainable travel experiences like trekking, hiking, and nature adventures.  
It promotes a green lifestyle and connects people with nature through meaningful experiences.

---

## 🚀 Live Demo
https://eco-adventure26.netlify.app/

---

## ✨ Features

- 🌍 Eco-friendly adventure listings
- 🔐 User authentication (Firebase)
- 👤 User profile system
- 📱 Fully responsive design
- ⚡ Fast performance with React
- 🧭 Clean and modern UI (Tailwind CSS / DaisyUI)

---

## 🛠️ Tech Stack

- React.js
- React Router DOM
- Firebase Authentication
- Tailwind CSS
- DaisyUI
- Netlify (Hosting)

---

## 📦 Installation

```bash
# Clone the repository
git clone https://github.com/your-username/ecoadventure.git

# Go to project folder
cd ecoadventure

# Install dependencies
npm install

# Run project
npm run dev
